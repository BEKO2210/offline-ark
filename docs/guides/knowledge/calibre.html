Calibre — eBook-Bibliothek offline verwalten

Wissen Sichern 15–20 Minuten
Windows Linux macOS
Fortschritt0%

Was ist das?

Für Kinder erklärt

Calibre ist wie ein riesiges Bücherregal auf deinem Computer. Du kannst alle deine digitalen Bücher dort hineinstellen, sortieren und lesen — ganz ohne Internet!

Einfach erklärt

Calibre ist ein kostenloses Programm, mit dem du deine eBooks verwalten, zwischen verschiedenen Formaten umwandeln und direkt am Computer lesen kannst. Es funktioniert wie eine persönliche digitale Bibliothek — komplett offline. Du kannst Bücher sortieren, Cover und Beschreibungen automatisch herunterladen und deine Sammlung sogar im lokalen Netzwerk mit anderen teilen.

Technische Details

Calibre ist in Python geschrieben und unterstützt über 20 eBook-Formate, darunter EPUB, PDF, MOBI, AZW3, FB2, LIT, DJVU, CBZ, CBR und viele mehr. Es bietet einen integrierten Format-Konverter, der nahezu jedes Format in jedes andere umwandeln kann. Die Metadaten-Verwaltung nutzt Online-Quellen wie Google Books, Amazon und Open Library. Der eingebaute Content Server (calibre-server) stellt die Bibliothek über HTTP bereit, sodass jedes Gerät im lokalen Netzwerk darauf zugreifen kann. Calibre speichert alle Bücher in einer strukturierten Ordnerhierarchie mit einer SQLite-Datenbank als Katalog.

Installation

1

Calibre herunterladen

Gehe zu calibre-ebook.com und lade den Windows-Installer herunter. Die Datei ist ca. 150 MB.

2

Installieren

Starte die heruntergeladene .msi-Datei und folge dem Installer. Die Standardeinstellungen sind perfekt. Calibre wird automatisch im Startmenü verknüpft.

3

Ersteinrichtung

Beim ersten Start wählt Calibre automatisch einen Bibliotheksordner (z.B. C:\Users\DeinName\Calibre Library). Du kannst den Speicherort jederzeit ändern.

1

Calibre installieren (Paketmanager)

Terminal
sudo apt install calibre

Alternativ den offiziellen Binary-Installer verwenden (empfohlen für die neueste Version):

Terminal
sudo -v && wget -nv -O- https://download.calibre-ebook.com/linux-installer.sh | sudo sh /dev/stdin
2

Calibre starten

Terminal
calibre &
1

Calibre installieren

Lade die DMG-Datei von calibre-ebook.com herunter oder nutze Homebrew:

Terminal
brew install --cask calibre
2

Ersteinrichtung

Öffne Calibre aus dem Programme-Ordner. Beim ersten Start wählst du den Bibliotheksordner. Die App funktioniert genauso wie auf Windows und Linux.

Bücher hinzufügen

1

Per Drag & Drop

Ziehe eBook-Dateien einfach per Drag & Drop in das Calibre-Hauptfenster. Die Bücher werden automatisch in die Bibliothek importiert und nach Autor sortiert.

2

Ordner importieren

Klicke auf "Bücher hinzufügen" in der Symbolleiste und wähle "Aus einem Ordner importieren". Calibre durchsucht den Ordner rekursiv nach allen eBook-Dateien.

Terminal
# Bücher über die Kommandozeile hinzufügen
calibredb add /pfad/zum/buch.epub
# Ganzen Ordner rekursiv importieren
calibredb add -r /pfad/zum/ordner/
3

Metadaten bearbeiten

Wähle ein Buch aus und klicke auf "Metadaten bearbeiten". Du kannst Titel, Autor, Beschreibung und Tags manuell anpassen — oder klicke auf "Metadaten herunterladen", um sie automatisch aus dem Internet zu laden (Cover, Beschreibung, ISBN etc.).

Tipp: Du kannst Metadaten für mehrere Bücher gleichzeitig herunterladen: Wähle alle Bücher aus, dann "Metadaten herunterladen". Calibre erledigt den Rest im Hintergrund.

1

Buch konvertieren (GUI)

Wähle ein Buch aus, klicke auf "Bücher konvertieren" in der Symbolleiste. Wähle das Zielformat (z.B. EPUB) und klicke auf "OK". Die Konvertierung läuft im Hintergrund.

2

Buch konvertieren (Kommandozeile)

Terminal
# EPUB zu PDF konvertieren
ebook-convert buch.epub buch.pdf

# MOBI zu EPUB mit benutzerdefinierten Einstellungen
ebook-convert buch.mobi buch.epub --output-profile tablet

# Massenkonvertierung eines ganzen Ordners
calibredb add -r /pfad/zum/ordner/ && calibredb catalog katalog.epub

Content Server

Mit dem eingebauten Content Server kannst du deine Bibliothek im lokalen Netzwerk teilen. Jedes Gerät mit einem Browser kann dann auf deine Bücher zugreifen.

1

Server über die GUI starten

Klicke in Calibre auf "Verbinden/Teilen" und dann auf "Content Server starten". Der Server ist standardmäßig unter http://localhost:8080 erreichbar.

2

Server über die Kommandozeile starten

Terminal
# Content Server starten
calibre-server --port 8080 /pfad/zur/bibliothek

# Mit Benutzerauthentifizierung
calibre-server --port 8080 --enable-auth /pfad/zur/bibliothek

# Benutzer hinzufügen
calibre-server --manage-users
3

Auf die Bibliothek zugreifen

Öffne auf einem beliebigen Gerät im Netzwerk den Browser und navigiere zu http://deine-ip:8080. Du kannst Bücher durchsuchen, lesen und herunterladen.

Sicherheitshinweis: Der Content Server sollte nur im lokalen Netzwerk verwendet werden. Aktiviere die Authentifizierung, wenn andere Personen Zugang zu deinem Netzwerk haben.

E-Book Reader

Calibre hat einen leistungsstarken integrierten eBook-Viewer, der EPUB, PDF, MOBI und viele weitere Formate direkt anzeigen kann.

1

Buch im Viewer öffnen

Doppelklicke auf ein Buch in der Bibliothek oder wähle es aus und klicke auf "Bücher anzeigen". Der Viewer öffnet sich in einem eigenen Fenster.

2

Viewer-Funktionen

Der integrierte Viewer bietet:

  • Anpassbare Schriftgröße, Schriftart und Hintergrundfarbe
  • Lesezeichen und Markierungen
  • Volltextsuche im gesamten Buch
  • Inhaltsverzeichnis-Navigation
  • Nachtmodus für augenschonendes Lesen
3

Viewer über die Kommandozeile starten

Terminal
# eBook direkt im Viewer öffnen
ebook-viewer /pfad/zum/buch.epub

Konfiguration

1

Bibliotheksordner ändern

Gehe zu "Einstellungen" und klicke auf "Bibliothek verschieben/kopieren". Du kannst die Bibliothek auf eine externe Festplatte oder ein NAS verschieben — ideal für Backups.

Standardpfade
# Windows
C:\Users\DeinName\Calibre Library

# Linux
~/Calibre Library

# macOS
~/Calibre Library
2

Metadaten-Download konfigurieren

Unter "Einstellungen" > "Metadaten-Download" kannst du festlegen, welche Online-Quellen Calibre für Cover und Buchinfos nutzen soll (Google Books, Amazon, Open Library etc.). Diese Metadaten werden lokal gespeichert und stehen danach offline zur Verfügung.

3

Cover-Bilder verwalten

Calibre kann Cover automatisch aus dem Internet herunterladen oder du kannst eigene Cover-Bilder per Drag & Drop zuweisen. Unter "Einstellungen" > "Aussehen" kannst du die Covergröße und das Anzeigeformat anpassen.

Tipp: Lade Metadaten und Cover herunter, solange du noch Internet hast. Danach funktioniert deine Bibliothek vollständig offline!

Deine Schritte

  • Calibre installiert
  • Erste Bücher zur Bibliothek hinzugefügt
  • Metadaten und Cover heruntergeladen
  • Ein Buch erfolgreich konvertiert
  • E-Book im integrierten Viewer gelesen
  • Content Server getestet (optional)

Geschafft!

Du hast jetzt deine eigene digitale Bibliothek — komplett offline. Bücher verwalten, konvertieren, lesen und im Netzwerk teilen, alles ohne Internet.

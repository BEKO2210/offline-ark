Kiwix — Wikipedia offline verfügbar machen

Wissen Sichern 15–30 Minuten
Windows Linux macOS Android iOS Raspberry Pi
Fortschritt0%

Was ist das?

Für Kinder erklärt

Kiwix ist wie eine Zauberkiste, die das ganze Wikipedia auf deinen Computer packt — du kannst alles nachschlagen, auch wenn kein Internet da ist!

Einfach erklärt

Kiwix ist ein kostenloses Programm, mit dem du die komplette Wikipedia (und andere Webseiten) auf deinen Computer, dein Handy oder einen USB-Stick herunterladen kannst. Danach kannst du alles nachschlagen — auch wenn du gar kein Internet hast. Stell es dir vor wie eine riesige Enzyklopädie, die auf einen Stick passt.

Technische Details

Kiwix nutzt das ZIM-Dateiformat (Zeno IMproved), ein hochkomprimiertes Archivformat für Web-Inhalte. Die deutsche Wikipedia (mit Bildern) ist ca. 50 GB groß, ohne Bilder ca. 10 GB. Kiwix bietet einen integrierten HTTP-Server (kiwix-serve), mit dem du Inhalte im lokalen Netzwerk teilen kannst. Verfügbare Inhalte gehen weit über Wikipedia hinaus: Stack Overflow, Project Gutenberg, TED Talks, Wikivoyage und mehr.

Installation

1

Kiwix herunterladen

Gehe zu kiwix.org und klicke auf "Download Kiwix". Wähle die Windows-Version. Die App ist klein (ca. 80 MB).

2

Installieren

Starte die heruntergeladene Datei und folge dem Installer. Standardeinstellungen sind perfekt.

3

Wikipedia herunterladen

Öffne Kiwix und klicke auf das "+" Symbol oder "Bibliothek". Du siehst eine Liste aller verfügbaren Inhalte. Suche nach "Wikipedia (de)" und klicke auf "Download".

Dateigröße beachten: Wikipedia Deutsch mit Bildern ist ca. 50 GB. Ohne Bilder ca. 10 GB. Stelle sicher, dass du genug Speicherplatz hast!

4

Offline lesen!

Nach dem Download kannst du sofort suchen und lesen — ganz ohne Internet. Die Suche funktioniert genauso wie auf der echten Wikipedia.

1

Kiwix installieren

Am einfachsten über Flatpak:

Terminal
flatpak install flathub org.kiwix.desktop

Alternativ kannst du das AppImage von kiwix.org herunterladen.

2

Kiwix-Tools für den Server-Modus

Terminal
sudo apt install kiwix-tools
3

Wikipedia im Netzwerk teilen (optional)

Terminal
kiwix-serve wikipedia_de.zim -p 8080

Öffne dann im Browser: http://localhost:8080. Jedes Gerät in deinem Netzwerk kann darauf zugreifen!

1

Kiwix installieren

Lade Kiwix von kiwix.org herunter oder nutze Homebrew:

Terminal
brew install --cask kiwix
2

Inhalte herunterladen

Öffne Kiwix, gehe zur Bibliothek und lade die gewünschten Inhalte herunter. Die App funktioniert genauso wie auf Windows.

1

Kiwix installieren

Installiere Kiwix aus dem Google Play Store oder von F-Droid (für mehr Privatsphäre). Suche einfach nach "Kiwix".

2

Wikipedia herunterladen

Öffne die App, tippe auf "Bibliothek" und wähle die gewünschte Wikipedia-Version. Für Handys empfehlen wir die Version ohne Bilder (~10 GB), um Speicherplatz zu sparen.

Tipp: Wenn du eine SD-Karte hast, kannst du in den App-Einstellungen den Speicherort ändern!

1

Kiwix installieren

Installiere Kiwix aus dem App Store. Die App ist kostenlos.

2

Inhalte laden

Öffne die App und lade Inhalte direkt in der App herunter. Beachte, dass iOS-Speicher meist begrenzt ist — wähle eine kleinere Wikipedia-Version oder einzelne Themengebiete.

1

Kiwix-Tools installieren

Terminal
sudo apt update
sudo apt install kiwix-tools
2

ZIM-Datei herunterladen

Lade die gewünschte ZIM-Datei auf eine externe Festplatte oder eine große SD-Karte:

Terminal
# Beispiel: Wikipedia Deutsch (ohne Bilder, ~10 GB)
wget https://download.kiwix.org/zim/wikipedia/wikipedia_de_all_nopic.zim
3

Als Server starten

Terminal
kiwix-serve /pfad/zur/wikipedia_de.zim -p 8080

Jetzt kann jedes Gerät in deinem WLAN über http://raspberry-pi-ip:8080 auf Wikipedia zugreifen!

4

Autostart einrichten

Terminal
sudo tee /etc/systemd/system/kiwix.service <<EOF
[Unit]
Description=Kiwix Server
After=network.target

[Service]
ExecStart=/usr/bin/kiwix-serve /pfad/zur/wikipedia_de.zim -p 8080
Restart=always
User=pi

[Install]
WantedBy=multi-user.target
EOF

sudo systemctl enable kiwix
sudo systemctl start kiwix

Jetzt startet der Kiwix-Server automatisch bei jedem Neustart deines Raspberry Pi!

LocalSend — Dateien teilen ohne Internet

Kommunikation 5–10 Minuten
Windows Linux macOS Android iOS
Fortschritt0%

Was ist das?

Für Kinder erklärt

Stell dir vor, du willst ein Foto von deinem Handy an den Computer schicken — ohne Kabel, ohne Internet, ohne E-Mail. LocalSend ist wie ein unsichtbares Kabel zwischen deinen Geräten. Du wählst die Datei aus, tippst auf Senden, und schwupps ist sie auf dem anderen Gerät!

Einfach erklärt

LocalSend ist eine kostenlose, quelloffene App, mit der du Dateien, Fotos, Videos und Text direkt zwischen deinen Geräten austauschen kannst — ganz ohne Internet. Es funktioniert wie Apples AirDrop, aber auf allen Plattformen: Windows, Linux, macOS, Android und iOS. Die Geräte müssen sich nur im selben WLAN (oder lokalen Netzwerk) befinden. Kein Konto nötig, keine Cloud, keine Datenweitergabe an Dritte.

Technische Details

LocalSend nutzt ein eigenes REST-basiertes Protokoll zur Geräteerkennung und Dateiübertragung über das lokale Netzwerk. Die Kommunikation wird mit TLS/HTTPS verschlüsselt — jedes Gerät generiert beim ersten Start ein selbstsigniertes Zertifikat. Die Erkennung anderer Geräte erfolgt via UDP-Multicast auf Port 53317. Dateien werden direkt Peer-to-Peer übertragen, ohne zentralen Server. Die App ist in Flutter/Dart geschrieben und vollständig Open Source (MIT-Lizenz). Der Quellcode ist auf GitHub unter localsend/localsend einsehbar.

Installation

1

LocalSend herunterladen

Gehe zu localsend.org und klicke auf den Download-Button für Windows. Alternativ kannst du die App auch über den Microsoft Store oder das Terminal installieren:

Terminal (PowerShell)
winget install LocalSend.LocalSend
2

Installieren und starten

Führe den Installer aus und folge den Anweisungen. Nach der Installation starte LocalSend. Die App zeigt sofort deinen Gerätenamen und sucht nach anderen Geräten im Netzwerk.

3

Firewall-Zugriff erlauben

Beim ersten Start fragt Windows nach einer Firewall-Berechtigung. Klicke auf "Zugriff zulassen" — das ist wichtig, damit andere Geräte dich finden können. Erlaube den Zugriff für private Netzwerke.

Firewall-Tipp: Falls die Abfrage nicht erscheint oder du sie weggeklickt hast: Öffne Windows-Sicherheit → Firewall → App durch Firewall zulassen und aktiviere LocalSend für private Netzwerke.

1

LocalSend installieren

Terminal
# Flatpak (empfohlen — funktioniert auf allen Distributionen)
flatpak install flathub org.localsend.localsend_app

# Snap
sudo snap install localsend

# Arch Linux (AUR)
yay -S localsend-bin

Alternativ kannst du das AppImage oder .deb/.rpm-Paket von localsend.org oder der GitHub-Releases-Seite herunterladen.

2

Starten

Starte LocalSend über das Anwendungsmenü oder im Terminal:

Terminal
flatpak run org.localsend.localsend_app
3

Firewall konfigurieren (falls nötig)

Falls andere Geräte nicht sichtbar sind, muss Port 53317 (TCP und UDP) in der Firewall freigegeben werden:

Terminal
# UFW (Ubuntu/Debian)
sudo ufw allow 53317

# firewalld (Fedora)
sudo firewall-cmd --add-port=53317/tcp --permanent
sudo firewall-cmd --add-port=53317/udp --permanent
sudo firewall-cmd --reload
1

LocalSend installieren

Lade LocalSend von localsend.org herunter, oder nutze Homebrew:

Terminal
brew install --cask localsend

Die App ist auch im Mac App Store verfügbar.

2

Berechtigungen erteilen

Beim ersten Start fragt macOS, ob LocalSend Netzwerkverbindungen empfangen darf. Klicke auf "Erlauben". Falls du die Meldung verpasst hast, gehe zu Systemeinstellungen → Datenschutz & Sicherheit → Firewall → Optionen und erlaube eingehende Verbindungen für LocalSend.

1

LocalSend installieren

Installiere LocalSend aus dem Google Play Store oder aus F-Droid. Suche einfach nach "LocalSend". Die App ist kostenlos und werbefrei.

2

App öffnen

Starte die App. LocalSend erkennt automatisch andere Geräte im selben WLAN. Dein Gerät bekommt einen zufälligen Namen (z.B. "Smaragd-Falke"), den du in den Einstellungen ändern kannst.

3

Berechtigungen

Erlaube LocalSend den Zugriff auf Dateien und Fotos, wenn du dazu aufgefordert wirst. Ohne diese Berechtigung kann die App keine Dateien senden oder empfangen.

1

LocalSend installieren

Öffne den App Store und suche nach "LocalSend". Tippe auf "Laden". Die App ist kostenlos.

2

App starten und erlauben

Starte LocalSend und erlaube den Zugriff auf das lokale Netzwerk, wenn iOS dich fragt. Das ist notwendig, damit die App andere Geräte finden kann. Erlaube außerdem den Zugriff auf Fotos und Dateien, wenn du diese teilen möchtest.

Dateien senden und empfangen

1

Beide Geräte ins selbe Netzwerk bringen

Stelle sicher, dass beide Geräte mit demselben WLAN verbunden sind. Das funktioniert auch über ein Ethernet-Kabel oder einen Hotspot — Hauptsache, die Geräte sind im selben lokalen Netzwerk.

2

LocalSend auf beiden Geräten öffnen

Starte LocalSend auf dem sendenden und dem empfangenden Gerät. Nach wenigen Sekunden sollten sich die Geräte gegenseitig auf dem Startbildschirm anzeigen.

3

Datei(en) auswählen und senden

Auf dem sendenden Gerät:

  1. Tippe auf das Zielgerät in der Liste
  2. Wähle Dateien, Fotos, Videos oder Text aus
  3. Tippe auf "Senden"

Auf dem empfangenden Gerät erscheint eine Anfrage. Tippe auf "Akzeptieren" — fertig! Die Dateien werden direkt übertragen.

4

Alternative: Teilen-Menü nutzen

Auf Mobilgeräten kannst du Dateien auch direkt über das System-Teilen-Menü senden. Öffne ein Foto oder eine Datei, tippe auf "Teilen" und wähle LocalSend aus der Liste der Apps.

Tipps für eine reibungslose Übertragung
  • Geräte nicht sichtbar? Prüfe, ob beide Geräte im selben WLAN sind. Manche Router isolieren Geräte voneinander ("Client Isolation" / "AP Isolation") — das muss deaktiviert sein.
  • Firewall: Auf Desktop-Betriebssystemen muss LocalSend durch die Firewall zugelassen werden (Port 53317 TCP/UDP).
  • Schnell-Modus: In den Einstellungen kannst du "Schnellspeichern" aktivieren — dann werden eingehende Dateien automatisch ohne Bestätigung angenommen. Nur in vertrauenswürdigen Netzwerken verwenden!
  • Ohne WLAN: Du kannst auch einen Handy-Hotspot als Netzwerk nutzen. Verbinde das zweite Gerät mit dem Hotspot — Internetverbindung ist dabei nicht nötig.
  • Gerätename ändern: Unter Einstellungen kannst du deinen Gerätenamen anpassen, um Geräte leichter zu unterscheiden.

Deine Schritte

  • LocalSend auf dem ersten Gerät installiert
  • LocalSend auf dem zweiten Gerät installiert
  • Beide Geräte im selben WLAN
  • Geräte erkennen sich gegenseitig
  • Erste Datei erfolgreich gesendet
  • Erste Datei erfolgreich empfangen

Geschafft!

Du kannst jetzt Dateien zwischen all deinen Geräten teilen — ohne Internet, ohne Cloud, ohne Konto. Deine Daten bleiben in deinem Netzwerk und werden verschlüsselt übertragen. LocalSend ist die perfekte Alternative zu AirDrop, die auf allen Plattformen funktioniert.

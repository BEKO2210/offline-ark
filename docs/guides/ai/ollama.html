Ollama — LLMs lokal betreiben

KI Offline ⭐⭐ 30–60 Minuten
Windows Linux macOS
Fortschritt0%

📖 Was ist das?

👶
Für Kinder erklärt

Ollama ist wie ein schlauer Roboter-Freund, der auf deinem Computer wohnt und dir Fragen beantworten kann — auch wenn das Internet aus ist!

👵
Einfach erklärt

Ollama ist ein Programm, mit dem du eine künstliche Intelligenz direkt auf deinem Computer laufen lassen kannst — ganz ohne Internet. Du kannst der KI Fragen stellen, dir Texte schreiben lassen oder Ideen sammeln. Stell es dir vor wie eine digitale Enzyklopädie, die auch Fragen beantworten kann. Alles bleibt privat auf deinem Rechner.

🔧 Technische Details

Ollama ist ein Inference-Server für quantisierte Large Language Models (LLMs) im GGUF-Format. Er bietet eine REST-API, automatische GPU-Beschleunigung (CUDA für NVIDIA, ROCm für AMD, Metal für Apple Silicon), Modellverwaltung mit Pull/Push, parallele Anfragen und ein CLI-Interface. Unterstützt werden Modelle wie Llama 3, Mistral, Gemma, Phi und viele weitere.

📥 Installation

1

Ollama herunterladen

Öffne deinen Browser und gehe zu ollama.com. Klicke auf den großen "Download for Windows"-Button. Die Datei ist ca. 90 MB groß.

2

Installer ausführen

Doppelklicke auf die heruntergeladene OllamaSetup.exe. Klicke "Installieren" und warte bis die Installation abgeschlossen ist. Ollama startet automatisch im Hintergrund (du siehst ein kleines Lama-Icon unten rechts in der Taskleiste).

⚠️ Häufige Fehler

Windows Defender warnt: Klicke auf "Weitere Informationen" und dann "Trotzdem ausführen". Ollama ist sicher und Open Source.

Firewall-Warnung: Erlaube den Zugriff. Ollama braucht lokale Netzwerkports (Standard: 11434).

3

PowerShell öffnen

Drücke Win + X und wähle "Terminal" oder "PowerShell". Alternativ: Drücke Win, tippe powershell und drücke Enter.

4

Dein erstes KI-Modell herunterladen

Tippe folgenden Befehl ein und drücke Enter:

PowerShell 
ollama pull llama3.2

Das Modell ist ca. 2 GB groß. Der Download dauert je nach Internetgeschwindigkeit 2–10 Minuten. Danach brauchst du kein Internet mehr!

5

Mit der KI chatten!

PowerShell 
ollama run llama3.2

Du kannst jetzt Fragen stellen! Tippe zum Beispiel: Erkläre mir was künstliche Intelligenz ist. Mit /bye beendest du den Chat.

1

Ollama installieren

Öffne ein Terminal und führe den Installationsbefehl aus:

Terminal 
curl -fsSL https://ollama.com/install.sh | sh

Das Skript erkennt automatisch deine Linux-Distribution und installiert Ollama mit allen Abhängigkeiten.

2

Installation prüfen

Terminal
ollama --version

Du solltest eine Versionsnummer sehen, z.B. ollama version 0.5.x.

3

Modell herunterladen

Terminal
ollama pull llama3.2
4

Chatten!

Terminal
ollama run llama3.2
5

Dienst-Status prüfen (optional)

Terminal
sudo systemctl status ollama

Ollama läuft als systemd-Dienst und startet automatisch mit deinem Computer.

⚠️ Häufige Fehler

GPU nicht erkannt: Stelle sicher, dass NVIDIA-Treiber installiert sind: nvidia-smi

Permission denied: Dein Benutzer muss in der ollama-Gruppe sein. Führe aus: sudo usermod -aG ollama $USER und melde dich neu an.

1

Ollama installieren

Lade Ollama von ollama.com herunter, oder nutze Homebrew:

Terminal
brew install ollama
⚠️ Gatekeeper-Warnung

Beim ersten Start fragt macOS, ob du die App wirklich öffnen willst. Gehe zu Systemeinstellungen → Datenschutz & Sicherheit und klicke auf "Trotzdem öffnen".

2

Ollama starten

Starte die Ollama-App aus dem Launchpad oder öffne Terminal und starte den Dienst:

Terminal
ollama serve

Auf Apple Silicon (M1/M2/M3/M4) wird die GPU automatisch via Metal genutzt — das ist deutlich schneller als CPU!

3

Modell herunterladen und chatten

Terminal
ollama pull llama3.2
ollama run llama3.2

📊 Welches Modell für deinen Computer?

Die Modellgröße bestimmt die Qualität der Antworten. Wähle ein Modell das zu deinem Arbeitsspeicher (RAM) passt:

RAMModellBefehlQualität
4 GBPhi-3 Miniollama pull phi3Gut für einfache Aufgaben
8 GBLlama 3.2 (3B)ollama pull llama3.2Sehr gut, empfohlen als Einstieg
16 GBLlama 3.1 (8B)ollama pull llama3.1Exzellent für die meisten Aufgaben
32 GB+Llama 3.1 (70B)ollama pull llama3.1:70bNahezu ChatGPT-Niveau
8 GBMistral (7B)ollama pull mistralSchnell, gut für Code & Chat
8 GBGemma 2 (9B)ollama pull gemma2Google-Modell, stark bei Deutsch
💡

Tipp: Du weißt nicht wie viel RAM du hast? Windows: Strg+Shift+Esc → Reiter "Leistung". Linux: free -h. macOS: Apple-Menü → "Über diesen Mac".

⚙️ Die API nutzen (für Fortgeschrittene)

Ollama bietet eine REST-API auf Port 11434. Damit kannst du die KI in eigene Programme einbinden:

Terminal / cURL
curl http://localhost:11434/api/generate -d '{
  "model": "llama3.2",
  "prompt": "Erkläre mir was Photosynthese ist",
  "stream": false
}'

Modelle verwalten

Terminal
# Alle installierten Modelle anzeigen
ollama list

# Modell-Details anzeigen
ollama show llama3.2

# Modell löschen (um Speicherplatz freizugeben)
ollama rm llama3.2

💡 GPU vs CPU — Was ist schneller?

GPU (Grafikkarte): 5–20x schneller als CPU. Ollama erkennt NVIDIA-GPUs automatisch (CUDA). AMD-GPUs gehen über ROCm. Apple Silicon nutzt Metal.

CPU: Funktioniert immer, aber langsamer. Für kleine Modelle (unter 4B Parameter) trotzdem ausreichend schnell.

Empfehlung: Wenn du eine NVIDIA-Grafikkarte mit 6+ GB VRAM hast, nutze sie! Installiere die neuesten NVIDIA-Treiber.

💡 Was ist Quantisierung?

Quantisierung macht KI-Modelle kleiner und schneller, mit minimalem Qualitätsverlust. Stell es dir vor wie MP3 vs. WAV — kleiner, aber du merkst den Unterschied kaum.

Q4_K_M: Guter Kompromiss aus Qualität und Größe (Standard bei Ollama)

Q5_K_M: Etwas besser, aber größer

Q8_0: Nahezu verlustfrei, aber deutlich größer

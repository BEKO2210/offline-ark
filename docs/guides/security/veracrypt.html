VeraCrypt — Festplattenverschlüsselung offline

Sicherheit 20–30 Minuten
Windows Linux macOS
Fortschritt0%

Was ist das?

Für Kinder erklärt

VeraCrypt ist wie ein unsichtbarer Tresor auf deinem Computer. Du kannst Dateien dort hineinlegen, und niemand kann sie sehen oder öffnen — nur du mit deinem geheimen Passwort!

Einfach erklärt

VeraCrypt ist eine kostenlose Open-Source-Software zur Festplattenverschlüsselung. Du kannst damit ganze Festplatten, Partitionen oder einzelne verschlüsselte Container erstellen. Ein Container ist wie ein virtueller Tresor — eine einzelne Datei, die sich als Laufwerk einbinden lässt. Ohne Passwort sieht niemand, was drin ist. VeraCrypt ist der Nachfolger von TrueCrypt und gilt als einer der sichersten Verschlüsselungstools weltweit.

Technische Details

VeraCrypt unterstützt mehrere Verschlüsselungsalgorithmen: AES-256, Serpent, Twofish — sowie Kaskaden-Kombinationen wie AES(Twofish(Serpent)) für maximale Sicherheit. Hash-Algorithmen: SHA-512, Whirlpool, SHA-256, Streebog. VeraCrypt nutzt PBKDF2 mit sehr hohen Iterationszahlen zur Key-Derivation, was Brute-Force-Angriffe extrem verlangsamt. Features: Hidden Volumes (versteckte Volumes mit plausibler Abstreitbarkeit), Full Disk Encryption (FDE), Rescue Disk, Keyfile-Unterstützung und Hardware-beschleunigte AES-NI-Verschlüsselung.

Installation

1

VeraCrypt herunterladen

Gehe zu veracrypt.fr und klicke auf "Downloads". Wähle den Windows-Installer (VeraCrypt Setup x.xx.x.exe). Die Datei ist ca. 35 MB.

2

Installieren

Führe den Installer aus. Wähle "Install" (nicht "Extract") für die volle Funktionalität inklusive Systemverschlüsselung. Alle Standardeinstellungen sind gut. Nach der Installation starte VeraCrypt.

3

Sprache auf Deutsch stellen

Gehe zu Settings → Language und wähle Deutsch. VeraCrypt startet danach in deutscher Sprache.

1

VeraCrypt installieren

Terminal
# Ubuntu / Debian — PPA hinzufügen
sudo add-apt-repository ppa:unit193/encryption
sudo apt update
sudo apt install veracrypt

# Oder manuell von veracrypt.fr herunterladen:
# .deb-Paket für Ubuntu/Debian
# .rpm-Paket für Fedora/openSUSE
# Generisches Installationsskript für andere Distributionen
2

Starten

Starte VeraCrypt über das Anwendungsmenü oder mit dem Befehl veracrypt im Terminal.

1

macFUSE installieren

VeraCrypt benötigt macFUSE (ehemals OSXFUSE). Lade es von osxfuse.github.io herunter und installiere es zuerst.

Terminal
# Alternativ via Homebrew:
brew install --cask macfuse
2

VeraCrypt installieren

Lade die DMG-Datei von veracrypt.fr herunter. Öffne die DMG und ziehe VeraCrypt in den Programme-Ordner.

Terminal
# Alternativ via Homebrew:
brew install --cask veracrypt

Hinweis: Auf neueren macOS-Versionen musst du macFUSE in den Systemeinstellungen unter Datenschutz & Sicherheit manuell erlauben. Ein Neustart kann erforderlich sein.

Verschlüsselten Container erstellen

Was ist ein Container? Ein Container ist eine einzelne verschlüsselte Datei, die sich wie ein eigenes Laufwerk (z.B. Laufwerk E:) einbinden lässt. Perfekt, um sensible Dokumente sicher aufzubewahren.

1

Volume erstellen

Klicke in VeraCrypt auf "Volume erstellen". Wähle "Eine verschlüsselte Containerdatei erstellen" und klicke auf "Weiter".

2

Volume-Typ wählen

Wähle "Standard-VeraCrypt-Volume". (Das "Versteckte Volume" ist für Fortgeschrittene — es bietet plausible Abstreitbarkeit.)

3

Speicherort wählen

Klicke auf "Datei..." und wähle einen Speicherort und Dateinamen für deinen Container, z.B. C:\Benutzer\DeinName\Dokumente\MeinTresor. Die Dateiendung ist egal — du kannst sie weglassen oder z.B. .hc verwenden.

4

Verschlüsselungsalgorithmus wählen

Empfohlene Einstellungen:

EinstellungEmpfehlung
VerschlüsselungAES (schnell und sicher) oder AES(Twofish(Serpent)) (maximal sicher)
Hash-AlgorithmusSHA-512
5

Größe festlegen

Wähle die Größe deines Containers. Beispiele:

  • 500 MB — für Dokumente und Passwort-Datenbanken
  • 5 GB — für Fotos und kleinere Projekte
  • 50 GB+ — für große Datenmengen
6

Passwort festlegen

Wähle ein starkes Passwort mit mindestens 20 Zeichen. Je länger, desto sicherer.

WARNUNG — Passwort vergessen = Daten verloren! Es gibt KEINE Möglichkeit, ein vergessenes VeraCrypt-Passwort wiederherzustellen. Kein Kundensupport, kein "Passwort zurücksetzen", kein Hintertürchen. Deine Daten sind dann unwiederbringlich verloren! Schreibe dein Passwort auf und bewahre es an einem sicheren Ort auf (z.B. in einem physischen Tresor oder in deinem Passwort-Manager).

7

Formatierung und Zufallsdaten

Wähle NTFS (Windows) oder exFAT (plattformübergreifend) als Dateisystem. Bewege nun die Maus möglichst zufällig im Fenster hin und her — das generiert kryptografisch sichere Zufallsdaten. Je länger, desto besser (mindestens 30 Sekunden). Klicke dann auf "Formatieren".

Container einbinden (mounten)

1

Laufwerksbuchstabe wählen

Wähle in der VeraCrypt-Hauptansicht einen freien Laufwerksbuchstaben (z.B. M:).

2

Container-Datei auswählen

Klicke auf "Datei..." und wähle deine Container-Datei aus.

3

Einbinden

Klicke auf "Einbinden", gib dein Passwort ein und warte kurz. Dein Container erscheint nun als eigenes Laufwerk im Datei-Explorer. Du kannst Dateien hinein- und herauskopieren wie bei einem normalen Laufwerk.

4

Trennen nicht vergessen!

Wenn du fertig bist, klicke in VeraCrypt auf "Trennen" (oder "Alle trennen"). Erst dann ist dein Container wieder sicher verschlossen. Schließe zuerst alle geöffneten Dateien aus dem Container!

USB-Stick verschlüsseln

Achtung: Beim Verschlüsseln einer Partition/eines USB-Sticks werden alle vorhandenen Daten gelöscht! Sichere vorher alle Daten.

1

Volume erstellen

Klicke auf "Volume erstellen" und wähle "Eine Partition / ein Laufwerk verschlüsseln".

2

USB-Stick auswählen

Klicke auf "Datenträger..." und wähle deinen USB-Stick aus. Achte genau darauf, das richtige Laufwerk zu wählen!

3

Verschlüsseln

Folge dem Assistenten wie bei der Container-Erstellung: Algorithmus wählen, Passwort setzen, Maus bewegen, formatieren. Der USB-Stick ist danach nur noch über VeraCrypt zugänglich.

4

USB-Stick nutzen

Zum Öffnen: Klicke in VeraCrypt auf "Datenträger...", wähle den USB-Stick, klicke "Einbinden" und gib dein Passwort ein. Zum Schließen: Erst "Trennen" in VeraCrypt, dann USB-Stick sicher entfernen.

Komplette Festplatte verschlüsseln (Full Disk Encryption)

Nur für Fortgeschrittene! Die Systemverschlüsselung ist ein mächtiges Feature, aber ein Fehler kann dazu führen, dass dein Computer nicht mehr startet. Erstelle vorher ein vollständiges Backup!

Mit VeraCrypt kannst du unter Windows deine komplette Systempartition verschlüsseln. Nach der Einrichtung musst du bei jedem Computerstart dein VeraCrypt-Passwort eingeben — noch bevor Windows lädt.

1

Systemverschlüsselung starten

Gehe zu System → Systempartition/Systemlaufwerk verschlüsseln.

2

Rettungsdatenträger erstellen

VeraCrypt erstellt zwingend einen Rettungsdatenträger (Rescue Disk). Dieser ist lebenswichtig — falls der Bootloader beschädigt wird, kannst du damit dein System wiederherstellen. Bewahre ihn sicher auf!

3

Vortest und Verschlüsselung

VeraCrypt führt zuerst einen Vortest durch — dein Computer startet neu und du musst dein Passwort eingeben. Wenn das klappt, beginnt die eigentliche Verschlüsselung im Hintergrund. Du kannst den Computer währenddessen normal benutzen.

Linux und macOS

Unter Linux wird für die Systemverschlüsselung in der Regel LUKS (dm-crypt) verwendet, das bereits bei der Installation eingerichtet werden kann. VeraCrypt eignet sich unter Linux hauptsächlich für Container und externe Datenträger.

Unter macOS bietet Apple mit FileVault eine eigene Systemverschlüsselung an. VeraCrypt wird auf dem Mac vor allem für Container und externe Laufwerke genutzt.

Maximale Sicherheit: Kaskaden-Verschlüsselung

Für maximale Sicherheit kannst du eine Kaskaden-Verschlüsselung wählen, z.B. AES(Twofish(Serpent)). Dabei werden deine Daten dreifach mit unterschiedlichen Algorithmen verschlüsselt. Das ist langsamer, aber selbst wenn ein Algorithmus gebrochen wird, bleiben deine Daten durch die anderen geschützt.

Deine Schritte

  • VeraCrypt installiert
  • Ersten verschlüsselten Container erstellt
  • Container erfolgreich eingebunden und getrennt
  • Testdateien im Container gespeichert
  • Passwort sicher notiert und aufbewahrt
  • USB-Stick verschlüsselt (optional)
  • Backup des Container-Passworts an sicherem Ort

Geschafft!

Deine Daten sind jetzt sicher verschlüsselt — selbst wenn dein Computer oder USB-Stick gestohlen wird, kann niemand ohne dein Passwort auf die Inhalte zugreifen. Militärgrade Verschlüsselung, komplett kostenlos und open-source.

KeePassXC — Passwort-Manager offline

Sicherheit 15–20 Minuten
Windows Linux macOS Android (KeePassDX) iOS (KeePassium)
Fortschritt0%

Was ist das?

Für Kinder erklärt

KeePassXC ist wie ein super-sicherer Tresor für alle deine Passwörter. Du brauchst dir nur EIN Passwort zu merken, und der Tresor merkt sich alle anderen!

Einfach erklärt

KeePassXC ist ein Programm, das alle deine Passwörter in einer verschlüsselten Datei auf deinem Computer speichert. Kein Internet nötig, keine Cloud, keine monatlichen Kosten. Du erstellst ein einziges starkes "Master-Passwort" — damit öffnest du deinen digitalen Tresor. Darin sind dann alle anderen Passwörter sicher aufbewahrt.

Technische Details

KeePassXC nutzt das KDBX4-Format mit AES-256 oder ChaCha20-Verschlüsselung. Key-Derivation erfolgt über Argon2id (empfohlen) oder AES-KDF. Features: TOTP-Unterstützung (Zwei-Faktor-Codes), YubiKey/OnlyKey-Integration, Browser-Integration via KeePassXC-Browser, Auto-Type, SSH-Agent-Integration, Anhänge und benutzerdefinierte Felder. Die Datenbank ist eine einzelne .kdbx-Datei, die einfach kopiert und gesichert werden kann.

Installation

1

KeePassXC herunterladen

Gehe zu keepassxc.org und klicke auf "Download". Wähle den Windows-Installer (.msi). Die Datei ist ca. 40 MB.

2

Installieren

Führe den Installer aus und folge den Anweisungen. Alle Standardeinstellungen sind gut. Nach der Installation starte KeePassXC.

3

Neue Datenbank erstellen

Klicke auf "Neue Datenbank erstellen". Gib ihr einen Namen (z.B. "Meine Passwörter") und wähle einen Speicherort — am besten in einem Ordner, den du regelmäßig sicherst.

4

Master-Passwort wählen

Wähle ein starkes Master-Passwort. Das ist das EINZIGE Passwort, das du dir merken musst!

Tipps für ein starkes Master-Passwort

Nutze einen Satz, den du dir gut merken kannst, z.B.: Mein-Hund-frisst-gerne-3-Bananen! — Lang, einprägsam, trotzdem sicher. Mindestens 12 Zeichen, am besten 20+.

1

KeePassXC installieren

Terminal
# Ubuntu / Debian
sudo apt install keepassxc

# Fedora
sudo dnf install keepassxc

# Oder als Flatpak (alle Distributionen)
flatpak install flathub org.keepassxc.KeePassXC
2

Datenbank erstellen

Starte KeePassXC und erstelle eine neue Datenbank — die Schritte sind identisch mit der Windows-Anleitung oben.

1

KeePassXC installieren

Lade KeePassXC von keepassxc.org herunter, oder nutze Homebrew:

Terminal
brew install --cask keepassxc
1

KeePassDX installieren

Installiere KeePassDX (nicht KeePassXC!) aus dem Google Play Store oder F-Droid. KeePassDX ist die beste Android-App für KeePass-Datenbanken.

2

Datenbank öffnen

Kopiere deine .kdbx-Datei auf dein Handy (via USB, Syncthing oder LocalSend) und öffne sie in KeePassDX.

1

KeePassium installieren

Installiere KeePassium aus dem App Store. Die Basis-Version ist kostenlos und reicht für die meisten Nutzer.

2

Datenbank öffnen

Übertrage deine .kdbx-Datei via AirDrop, LocalSend oder USB-Kabel auf dein iPhone/iPad und öffne sie in KeePassium.

KeePassXC nutzen

1

Ersten Eintrag anlegen

Klicke auf das "+" Symbol oder Strg+N. Fülle aus:

  • Titel: Name der Webseite/App (z.B. "E-Mail")
  • Benutzername: Dein Login-Name oder E-Mail
  • Passwort: Klicke auf das Würfel-Symbol, um ein sicheres Passwort zu generieren!
  • URL: Die Webadresse (z.B. https://mail.example.com)
2

Passwort-Generator nutzen

Der eingebaute Generator erstellt super-sichere Passwörter. Empfohlene Einstellungen:

EinstellungEmpfehlung
Länge20+ Zeichen (je länger, desto sicherer)
GroßbuchstabenJa
KleinbuchstabenJa
ZahlenJa
SonderzeichenJa

Passphrase-Modus: Alternativ kannst du auch "Passphrase" wählen — das generiert Wörter wie Korrekt-Pferd-Batterie-Heftklammer. Leichter zu merken und trotzdem sicher!

3

Browser-Integration einrichten

KeePassXC kann Passwörter direkt im Browser ausfüllen:

  1. Gehe in KeePassXC zu Einstellungen → Browser-Integration
  2. Aktiviere die Integration für deinen Browser (Firefox/Chrome)
  3. Installiere die Browser-Erweiterung "KeePassXC-Browser"
  4. Verbinde die Erweiterung mit KeePassXC (Button in der Erweiterung)

Ab jetzt werden Passwörter automatisch angeboten, wenn du dich auf einer Webseite einloggen willst!

4

Datenbank sichern

Deine .kdbx-Datei ist dein digitaler Tresor. Mache regelmäßig Kopien!

  • Kopie auf einem USB-Stick
  • Kopie auf einer externen Festplatte
  • Optional: Kopie auf einem zweiten Computer

Wichtig: Wenn du dein Master-Passwort vergisst und keine Schlüsseldatei hast, sind alle Passwörter unwiederbringlich verloren! Schreib dein Master-Passwort auf und bewahre es an einem sicheren Ort auf (z.B. in einem echten Tresor).

Empfohlene Einstellungen

EinstellungEmpfehlungWarum?
VerschlüsselungAES-256-CBCMilitärgrade Verschlüsselung, schnell
Key-DerivationArgon2idModernster Schutz gegen Brute-Force
Auto-Sperre5 MinutenSperrt die Datenbank automatisch bei Inaktivität
Zwischenablage leeren10 SekundenPasswörter werden nach dem Kopieren automatisch aus der Zwischenablage gelöscht
Schlüsseldatei als Extra-Sicherheit

Zusätzlich zum Master-Passwort kannst du eine Schlüsseldatei erstellen. Das ist eine kleine Datei, die du z.B. auf einem USB-Stick aufbewahrst. Ohne diese Datei kann niemand deine Datenbank öffnen — selbst wenn er dein Passwort kennt. Doppelte Sicherheit!

Erstelle eine Schlüsseldatei unter Datenbank → Datenbankeinstellungen → Sicherheit → Schlüsseldatei hinzufügen.

Deine Schritte

  • KeePassXC installiert
  • Neue Datenbank erstellt
  • Master-Passwort gewählt
  • Ersten Eintrag angelegt
  • Passwort-Generator getestet
  • Browser-Integration eingerichtet (optional)
  • Backup der Datenbank erstellt

Geschafft!

Deine Passwörter sind jetzt sicher verschlüsselt — komplett offline, ohne Cloud, ohne monatliche Kosten. Niemand außer dir kann auf deine Passwörter zugreifen.
